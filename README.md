# Üyz English Speaking Coach

Üyz English Speaking Coach, yetişkin ve kıdemli (Üyz / Third Age) öğrenciler için tasarlanmış, mobil öncelikli bir İngilizce konuşma pratiği uygulamasıdır.

## Özellikler / Features

- **Parallel Stories / Paralel Hikayeler**: İngilizce ve Türkçe yan yana okuma.
- **9 Aşamalı Metot**: Hikaye, Anahtar Kelimeler, Zihin Haritası, 3 Zaman, Yeniden İfade, Sıra Değişimi, Kelime Analizi, Hayal Etme ve Koç Modu.
- **AI Feedback**: Gemini AI tarafından sağlanan mantık ve dilbilgisi analizi.
- **Bilingual UI**: Tüm arayüz hem İngilizce hem Türkçe olarak sunulmaktadır.

## Kurulum / Setup

1. Repository'yi klonlayın.
2. `npm install` ile bağımlılıkları yükleyin.
3. `.env` dosyası oluşturun ve `VITE_GEMINI_API_KEY` değerini ekleyin.
4. `npm run dev` ile yerel sunucuyu başlatın.

## Dağıtım / Deployment (Netlify)

1. Projeyi GitHub'a yükleyin.
2. Netlify üzerinden yeni bir site bağlayın.
3. `VITE_GEMINI_API_KEY` environment variable'ını Netlify paneline ekleyin.
4. Build command: `npm run build`, Publish directory: `dist`.
